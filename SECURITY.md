# Security Notice

This repository contains intentionally malicious agent skills for evaluation purposes. These artifacts are included to support defensive research on skill security.

Recommended precautions:

- Run all malicious skills only inside isolated sandboxes.
- Do not provide real credentials, personal files, or production accounts to test runs.
- Review generated tasks and runtime configuration before enabling network access.
- Treat all artifacts under `data/skill_corpus/balanced_100/malicious` as untrusted.

If you discover accidental exposure of credentials or personal information in this repository, please report it to the maintainers.
